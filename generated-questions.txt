
============================================================
802.1X / Port-based Network Access Control (PNAC)
============================================================

1. [mcq_single] [Bloom 2]
   Q: What is the main goal of 802.1X Port-based Network Access Control (PNAC)?
   Options: To provide network access based on MAC address filtering. | To require authentication before granting full network access. | To encrypt all network traffic. | To prevent denial-of-service attacks.
   Answer: To require authentication before granting full network access.

2. [true_false] [Bloom 1]
   Q: True or False: 802.1X authentication occurs *after* a host has been granted full network access.
   Answer: False

3. [fill_blank] [Bloom 1]
   Q: In the context of 802.1X, the device requesting network access is referred to as the ________.
   Options: Authenticator | Supplicant | Authentication Server | RADIUS Client
   Answer: Supplicant

4. [mcq_single] [Bloom 1]
   Q: Which device validates authentication requests, issues authorizations, and performs accounting of security events in an 802.1X architecture?
   Options: Supplicant | Authenticator | Authentication Server | Network Switch
   Answer: Authentication Server

5. [true_false] [Bloom 2]
   Q: True or False: The authenticator directly validates the authentication requests from the supplicant.
   Answer: False

6. [mcq_single] [Bloom 2]
   Q: The Extensible Authentication Protocol (EAP) provides which functionality in 802.1X?
   Options: A framework for deploying multiple types of authentication methods. | Encryption of all network traffic. | Firewall services. | Intrusion Detection.
   Answer: A framework for deploying multiple types of authentication methods.

7. [fill_blank] [Bloom 1]
   Q: ________ allows the authenticator and authentication server to communicate authentication and authorization decisions.
   Options: SNMP | RADIUS | TCP/IP | EAPoL
   Answer: RADIUS

8. [true_false] [Bloom 2]
   Q: True or False: The switch is able to decrypt the EAP packet containing the supplicant's credentials before forwarding it to the authentication server.
   Answer: False

9. [mcq_single] [Bloom 3]
   Q: Why is 802.1X considered a more secure solution compared to restricting access based solely on MAC addresses?
   Options: MAC addresses are easier to manage. | MAC addresses are static and cannot be changed. | MAC addresses are prone to spoofing. | 802.1X does not require any configuration.
   Answer: MAC addresses are prone to spoofing.

10. [mcq_single] [Bloom 1]
   Q: Which protocol is used to encapsulate EAP communications over a LAN?
   Options: RADIUS | EAPoL | SNMP | TACACS+
   Answer: EAPoL

11. [true_false] [Bloom 1]
   Q: True or False: 802.1X can be implemented on both wired (LAN) and wireless (WLAN) networks.
   Answer: True

12. [mcq_single] [Bloom 3]
   Q: What happens to a host's network access when it first connects to an 802.1X-enabled switch port?
   Options: Full network access is immediately granted. | Only EAPoL traffic is allowed. | All network traffic is blocked. | The port is disabled.
   Answer: Only EAPoL traffic is allowed.

13. [mcq_single] [Bloom 4]
   Q: Consider a scenario where a user's laptop is attempting to connect to a corporate network using 802.1X. The authentication server is unavailable. What is the likely outcome?
   Options: The laptop will be granted full network access regardless. | The laptop will be denied network access. | The laptop will bypass 802.1X and connect directly. | The switch will automatically reboot.
   Answer: The laptop will be denied network access.

14. [true_false] [Bloom 5]
   Q: True or False: Disabling 802.1X on a switch port will generally improve network security.
   Answer: False

15. [mcq_single] [Bloom 1]
   Q: Which of the following is NOT a typical role involved in an 802.1X authentication process?
   Options: Supplicant | Authenticator | Authentication Server | DHCP Server
   Answer: DHCP Server

16. [mcq_single] [Bloom 4]
   Q: What is the relationship between EAP and RADIUS in an 802.1X implementation?
   Options: EAP encapsulates RADIUS. | RADIUS encapsulates EAP. | EAP is used for communication between the authenticator and authentication server, while RADIUS defines the authentication methods. | RADIUS is used for communication between the authenticator and authentication server, carrying the EAP data.
   Answer: RADIUS is used for communication between the authenticator and authentication server, carrying the EAP data.

17. [true_false] [Bloom 3]
   Q: True or False: If the supplicant fails the authentication process, the authenticator will grant limited network access, allowing only access to resources needed for remediation (e.g., a captive portal).
   Answer: True

18. [mcq_single] [Bloom 4]
   Q: In a network using 802.1X, how can you ensure that only authorized devices are allowed on the network, even if a user plugs in a different device?
   Options: By using MAC address filtering. | By requiring user authentication credentials for each device. | By disabling unused switch ports. | By enabling port security.
   Answer: By requiring user authentication credentials for each device.

19. [fill_blank] [Bloom 2]

20. [true_false] [Bloom 2]
   Q: True or False: 802.1X only provides authentication and does not provide any authorization or accounting functionalities.
   Answer: False

21. [mcq_single] [Bloom 5]
   Q: What is a potential vulnerability associated with solely relying on 802.1X for network security, and how might you mitigate it?
   Options: Weak password policies; enforce strong password policies and multi-factor authentication. | Denial-of-service attacks; implement rate limiting and traffic filtering. | MAC address spoofing; use static IP addressing. | Unencrypted traffic; deploy IPsec or TLS for encryption.

22. [mcq_single] [Bloom 2]
   Q: Which of the following best describes the role of the authenticator in an 802.1X exchange?

   Answer: To act as an intermediary, relaying authentication information between the supplicant and the authentication server.
